# QA_System_for_IOT
物聯網數據分析與應用的FinalProject

Answer Generation的具體作法 : [Cai-Pei-Xuan/DRCD-AG-BertForMaskedLM](https://github.com/Cai-Pei-Xuan/DRCD-AG-BertForMaskedLM)

## PIP 安裝 requirements.txt 的套件
```
pip install -r requirements.txt
```
### Demo
```
python main.py
```

## 環境需求
- python 3.6+
- pytorch 1.3+
- Flask 1.1.1
- jieba 0.42.1
- transformers 2.2+
- CUDA Version: 10.0
